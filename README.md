# ANAPI - Guichet Unique

Application de gestion des investissements pour l'Agence Nationale pour la Promotion des Investissements (ANAPI) - RDC

## Technologies
- Next.js 15
- React 19
- TypeScript
- Tailwind CSS v4
- Prisma
- PostgreSQL

## Installation

```bash
npm install
npm run dev
```

L'application sera disponible sur http://localhost:3500
